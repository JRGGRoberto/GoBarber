## Configurando estrutura

### Desenvolvido em: NodeJS

1. Executar os comandos

  ```
  yarn init -y
```

  ```
  yarn add express
```

2. Criar pasta e arquivos:

 src
 * app.js
 * routes.js
 * server.js


3. Para executar:
 
```
node src/server.js
```

 No browser:

```
http://localhost:3333
```
